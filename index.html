Predicting the Bioactivity of Compounds Targeting SARS-CoV-2 Using Machine Learning

By Ronnel Bermas, John Asis, and Janna Reyes ML Work

We applied machine learning to predict the bioactivity of chemical compounds targeting the SARS-CoV-2 3C-like protease (3CLpro). Using molecular descriptors and chemical fingerprints, we evaluated several ML models. Random Forest Regression achieved the best performance, with the lowest MSE and MAE and the highest R².
